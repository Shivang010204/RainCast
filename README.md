RainCast AI is a sophisticated weather analytics platform that bridges the gap between raw meteorological data and actionable industry insights. Built for the **Modern Web**, it leverages **Machine Learning** to provide tailored advice for Agriculture, Construction, and Standard users.

## üöÄ Key Features

* **Dual-Model ML Engine:** Uses a Random Forest Classifier for rain prediction and a Gradient Boosting Regressor for temperature post-processing.
* **Sector-Specific Logic:** Custom advisory system for Farmers (irrigation/soil) and Construction (crane safety/concrete pouring).
* **Crowdsourced Verification:** A "Red Zone" logic that flags weather anomalies based on real-time user reports and metadata-verified photo evidence.
* **Glassmorphism UI:** A high-tech, terminal-inspired interface designed for clarity and "Intel" aesthetic.
* **Admin Control Deck:** Secure portal for staff to manage data integrity and purge logs.
